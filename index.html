Secure Login System using Hash + Salt

Task Description

Your assignment is to implement a secure user login system using the principles of hashing and salting, as discussed in the theory section.

Requirements

  • Develop a user registration form that securely hashes and salts new user passwords.
  • Implement a login verification process that correctly authenticates users by re-hashing their input password with their stored salt.
  • Ensure that original passwords are never stored in plaintext.
  • Provide a simple in-memory or file-based storage mechanism for usernames, salts, and hashed passwords for demonstration purposes.

Deliverables

  • Source code for the complete login system (HTML, CSS, JavaScript/Python for backend logic).
  • A brief report (PDF or Markdown) detailing your implementation choices, security considerations, and how your system addresses common vulnerabilities.
